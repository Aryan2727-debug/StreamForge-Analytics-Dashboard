# StreamForge Analytics Dashboard

A real-time analytics dashboard for StreamForge, an OTT video streaming platform.

The dashboard visualizes user engagement, video consumption, session activity, platform usage, and playback analytics collected from the StreamForge Video Player.

---

## Overview

StreamForge Analytics Dashboard provides actionable insights into how users interact with the video player.

It consumes analytics data generated by the StreamForge Video Player and stored through the StreamForge API.

The dashboard enables monitoring of:

* User activity
* Video engagement
* Session analytics
* Platform adoption
* Device usage
* Environment distribution
* Recent playback events

---

## Features

### Dashboard KPIs

* Total Users
* Total Sessions
* Total Analytics Events
* Total Video Plays
* Total Video Completions

### Analytics Visualizations

* Event Breakdown Chart
* Top Videos Chart
* Platform Breakdown Chart
* Device Breakdown Chart
* Environment Breakdown Chart

### Event Monitoring

* Recent Events Table
* Auto Refresh every 30 seconds
* Real-time visibility into user actions


## Analytics Flow

1. User interacts with the video player.
2. Analytics events are generated.
3. Events are sent to the backend API.
4. Events are stored in MongoDB Atlas.
5. Dashboard APIs aggregate analytics data.
6. Dashboard visualizes insights through charts and tables.
